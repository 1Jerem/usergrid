Usergrid Standalone Server Launcher

A simple standalone Java app that provides a GUI for running the server as a
desktop app.

Creates a standalone double-clickable jar.

For command-line options or production use, try usergrid-standalone instead.

A pre-built version of this app is available at:

https://usergrid.s3.amazonaws.com/usergrid-launcher-0.0.1-SNAPSHOT.jar
